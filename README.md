# Cyclistic Bike-Share Analysis

## Project Overview

This project was completed as part of the Google Data Analytics Professional Certificate Capstone Project.

The objective was to analyze Cyclistic bike-share data and identify behavioral differences between annual members and casual riders.

## Business Task

Determine how annual members and casual riders use Cyclistic bikes differently and provide recommendations to increase memberships.

## Tools Used

- SQL
- Google BigQuery
- Power BI

## Data Cleaning

- Combined monthly datasets using UNION ALL
- Removed invalid ride records
- Created ride length variable
- Created weekday variable

## Key Findings

### Rider Type
- Members generated significantly more rides than casual riders.

### Ride Duration
- Casual riders had longer average ride durations.

### Riding Behaviour
- Members were more active during weekdays.
- Casual riders were more active on weekends.

### Bike Preference
- Usage between classic bikes and electric bikes was relatively balanced.

## Recommendations

1. Launch weekend promotions for casual riders.
2. Create commuter-focused membership packages.
3. Offer incentives for casual riders to upgrade to annual membership.
4. Promote electric bike membership plans.
